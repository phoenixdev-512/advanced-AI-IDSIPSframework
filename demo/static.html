Quick Start Guide

üöÄ Option 1: View This Demo (No Setup Required)

You're already viewing the demo! This page showcases all features without requiring Python installation.

üíª Option 2: Run Full Dashboard Locally

To run the complete interactive dashboard with real functionality:

# Clone the repository
git clone https://github.com/phoenixdev-512/advanced-AI-IDSIPSframework.git
cd advanced-AI-IDSIPSframework

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the demo (simulated mode - no hardware required)
python3 main.py full --mode simulated

# Access dashboard at: http://localhost:8050

üéØ Option 3: Deployment Package (Easiest)

Use the pre-built deployment package for fastest setup:

# Build deployment package
./build_deployment.sh

# Run quickstart
cd argus-deployment
./quickstart.sh    # Linux/Mac
# or
quickstart.bat     # Windows

# Access dashboard at: http://localhost:8050
